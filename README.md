# vgsassc
Sass compiler using go-libsass

```
vgsassc --help
Usage of vgsassc:
  -I string
    	Specify directory to use for resolving @import
  -m	Shorthand for -output-style=compressed and takes precedence
  -o string
    	Output file
  -output-style string
    	One of: nested, expanded, compact or compressed (default "nested")
```
